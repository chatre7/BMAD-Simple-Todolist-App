# Story 1.3 จัดเก็บข้อมูลโลคอล

As a ผู้ใช้ทั่วไป,
I want แอปบันทึกข้อมูลงานไว้ในเครื่องและแสดงผลต่อเมื่อกลับมาใช้งานใหม่,
so that ฉันใช้งานได้แม้ออฟไลน์และไม่สูญหายง่าย
